Typography

This page makes continued use of the Glitch project you remixed as part of CSS Basics. You should follow along and experiment with each of the CSS properties described on this page.

There are two main categories of CSS properties that affect text styles:

  • font styles that affect the size and look of the font
  • text layout styles that affect the spacing and layout of the text within its containing element

ü§î Nothing on this page will teach you how to change the font that text is displayed in. That‚Äôs totally possible in CSS, but will be covered after you are introduced to some more basic text styling.

Font styles

The following CSS properties work on almost all text elements (like headings, paragraphs, anchor elements).

font-size

In CSS Basics, font-size was one of the first CSS properties you encountered. A font-size value is made up of two parts: a number and a unit of measurement.

From print design, you may be used to setting text size in points or picas; while those units do exist in CSS, it makes more sense for web design to use measurements that exist on screens. Given that, to start with, you‚Äôll be setting font sizes in pixels (just like the earlier example).

üëâüèª It‚Äôs worth noting here that the size of text in design programs, such as Sketch or the Adobe Suite, does not perfectly match the size of text in web browsers ‚Äì even when you use the same unit in each. This is a consistent pain point in web design and, unfortunately, I can only tell you to be aware of it; there‚Äôs no solid fix.

The default font-size in most browsers is 16 pixels for paragraph text, but users can easily override this number. You should definitely keep 16-18 pixels in mind as a baseline for readability (though this is also impacted by screen size, as well as your choice of font, its color contrast and other factors).

text-transform

This property adjusts the capitalization rules for your text; there are four common values:

  • uppercase transforms your text to ALL CAPS
  • lowercase transforms your text to lower case
  • capitalize transforms your text to be Title Case, where the first letter of each word is capitalized
  • none prevents text transformation and the text will use whatever capitalization exists in the HTML document

text-decoration

This property is mainly used for making changes to the default underline on links, but it does offer some other stylistic effects.

text-decoration is the first example of a shorthand CSS property; it combines four related CSS properties into a single declaration:

  • text-decoration-color
  • text-decoration-line
  • text-decoration-style
  • text-decoration-thickness

Text layout styles

The following CSS properties affect the layout of text.

text-align

This property sets the alignment of any block-level element. It uses the following keyword values: center, right, left, justify.

‚ö†Ô∏è Note that the specification of block-level elements in this description. Setting text-align on an inline element, like b or a, has no effect if the layout of that element has not been changed from its default.
