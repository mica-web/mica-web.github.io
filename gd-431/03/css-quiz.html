CSS Quiz

In Brief

This is a short quiz to establish what you remember or know of CSS.

Purpose

Just like last week’s debugging HTML exercise, this quiz is meant to help you and I both evaluate what familiarity with CSS you’ve retained from previous classes. Based on how you do on this quiz, you may want to do the additional readings and activities under “Next Steps” below.

Share What You Know

While this quiz has right and wrong answers, your score will not affect your grade; this is for participation points only.

Next Steps

If this activity made you want to review some CSS, I highly recommend these lessons from the tutorial HTML & CSS Is Hard (but it doesn’t have to be) by Oliver James.

There are instructions to follow along with for each of these, and I’ve created a Glitch project for you to remix and use for this purpose.

  • Hello, CSS (est. reading time: 19min)
  • CSS Box Model (est. reading time: 20min)
  • CSS Selectors (est. reading time: 24min)
