Project 1: Typography Specimen

In Brief

Using your current knowledge of HTML and CSS, markup the content provided and create styles for the text.

Purpose

Demonstrate your understanding of basic HTML and CSS, specifically as it pertains to color and typography.

Process

  1. Follow the setup instructions
  2. Use your current knowledge of HTML to add markup to the copy provided.
  3. Do some research into web typography.
    • Share at least one link about web typography in the #gd2 Slack channel. Explain why you think this resource is useful in 1-2 sentences.
  4. Update project markup based on current knowledge of HTML and add CSS.
  5. Present project to colleagues.

Submission

  1. Create a pull request for code review and final submission.
    • Include PR template questions and your responses.
  2. Send me a link to the pull request via Slack DM by midnight EST on 10/2 (see in your timezone).

Requirements

  • Your assignment should use:
    • at least one web font
    • at least one text color
    • at last one background color
  • Your assignment will be viewed at a resolution of 1200 x 800.
